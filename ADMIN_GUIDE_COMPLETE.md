# Admin Operations Guide

## Fabric Master Management
*   **Visibility**: Use the toggles at the top of `Fabric Master` to hide categories you don't use (e.g., if you only trade in finished goods, hide Base Fabric).
*   **Bulk Actions**: Select multiple fabrics in the list to Perform Bulk Delete or Status Updates.
*   **Search**: Use the search bar to find fabrics by Name, HSN, or Design Number.

## Design Management
*   **Central Gallery**: View all uploaded designs in `Design Management`.
*   **Bulk Upload**: Use the "Bulk Image Upload" tool to upload 50+ images at once. The system attempts to read Design Numbers from filenames.
*   **Pending Info**: Check the "Pending Info" dashboard to find designs that need fabric mapping.

## Cost & Accounting
*   **Bill Import**: Regularly upload bills using the AI tool. This keeps your cost data accurate without manual data entry.
*   **Cost Dashboard**: Check this weekly to see spend distribution between Material (Suppliers) and Service (Job Workers).
*   **HSN**: Keep the HSN Master updated. If GST rates change, update them here, and all future entries will use the new rate.

## People Management
*   **Suppliers/Workers**: Keep contact details and GST numbers updated here. These appear in dropdowns across the system.
*   **Export**: You can export the supplier list to Excel for offline use.
