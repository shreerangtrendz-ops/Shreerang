# Admin Operations Manual

## Account Management
*   **New Employee**: Go to `Team Management`, invite user via email, assign role (e.g., "Store Manager").
*   **Role Change**: Edit user in `Team Management` to upgrade/downgrade permissions.

## Emergency Procedures
*   **System Outage**: Check Supabase Status page. If system is up but app is down, contact the Hosting Provider support.
*   **Data Corruption**: If mass accidental deletion occurs, initiate Point-in-Time Recovery (PITR) from the Supabase dashboard immediately (requires Pro plan).
