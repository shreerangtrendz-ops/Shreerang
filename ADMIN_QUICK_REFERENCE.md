# Admin Quick Reference

## Telegram Commands
*   `/start`: Check if bot is alive.
*   *(Custom)*: You can configure n8n to listen for keywords like `/stats` to reply with daily sales.

## Common Error Codes
*   **401 Unauthorized**: Token expired. Generate new Meta Access Token.
*   **400 Bad Request**: Invalid JSON body in HTTP Request node (check quotes/commas).
*   **131030**: Message template does not exist.

## Emergency Contacts
*   **Hostinger Support**: Hosting/DB issues.
*   **Meta Business Support**: WhatsApp API bans or limits.
