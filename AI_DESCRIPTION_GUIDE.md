# AI Description Generator Guide

## Overview
The AI tool helps generate professional, SEO-friendly descriptions for your fabrics automatically, saving time on data entry.

## How It Works
The AI analyzes the technical specifications you enter (Base, Weave, GSM, Finish) and constructs a readable paragraph.

## Usage
1. Fill in the **Base**, **GSM**, **Construction**, and **Finish** fields in the Fabric Form.
2. Click the **"AI Generate"** button (Wand icon) next to the Description field.
3. Review the generated text. You can edit it manually if needed.

## Example
**Input**:
- Base: Rayon
- GSM: 140
- Finish: Bio-wash
- Construction: Plain

**AI Output**:
*"Premium quality Rayon fabric with a weight of 140 GSM. Features a durable Plain weave construction and treated with Bio-wash for a superior soft hand-feel. Ideal for comfortable daily wear garments."*
