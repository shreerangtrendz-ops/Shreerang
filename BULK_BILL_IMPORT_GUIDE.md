# Bulk Bill Import Guide (AI Powered)

## Overview
The new Bulk Bill Import feature uses AI to extract data from your physical or digital bills and map them to your fabric inventory.

## How to Use

1.  **Navigate**: Go to `Fabric Master > Bulk Bill Import`.
2.  **Upload**: Select "Purchase Bill" (for raw material) or "Job Work Bill" (for processing). Upload the image/PDF.
3.  **Extraction**: The system will analyze the bill and extract:
    *   Vendor Name
    *   Date & Amount
    *   HSN Codes
    *   Description
4.  **Mapping**: 
    *   The system attempts to auto-match the description to your existing fabrics.
    *   If no match is found, use the dropdown to select the correct fabric manually.
5.  **Confirm**: Click "Confirm Mapping" to save the record.

## Benefits
*   **Speed**: No manual data entry for every bill.
*   **Accuracy**: Reduces typing errors.
*   **Cost Tracking**: Automatically feeds into the Cost Tracking Dashboard.
