# Bulk Import Glossary

*   **Base Fabric**: The raw material (grey cloth). Level 1 of hierarchy.
*   **Finish Fabric**: Processed base fabric (dyed/printed). Level 2.
*   **Fancy Finish Fabric**: Embellished fabric. Level 3.
*   **SKU (Stock Keeping Unit)**: A unique code identifier for each fabric.
*   **Short Code**: A 3-6 letter abbreviation for Base/Construction (e.g., COTT, POLY).
*   **GSM**: Grams per Square Meter. A measure of fabric weight/density.
*   **Mapping**: The process of linking Excel columns (e.g., "Material") to System fields (e.g., "Base").
*   **Validation**: The automated check for errors before import.
*   **Batch**: A group of records processed together.
*   **RLS**: Row Level Security (Database permission system).
*   **Greige**: Raw, unfinished fabric directly from the loom.
*   **RFD**: Ready for Dyeing (processed but not colored).
