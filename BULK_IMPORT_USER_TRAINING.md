# Bulk Import User Training Module

## Objectives
By the end of this training, users will be able to:
1.  Navigate the Bulk Import Wizard.
2.  Prepare clean Excel data files.
3.  Troubleshoot common validation errors.
4.  Understand the Fabric Hierarchy.

## Prerequisites
*   Basic Excel skills (Copy/Paste, Dropdowns).
*   Access to Admin Dashboard.

## Training Modules

### Module 1: The Basics (15 mins)
*   **Concept**: Why do we use Bulk Import?
*   **Hierarchy**: Base -> Finish -> Fancy.
*   **Demo**: Importing 5 Base Fabrics.

### Module 2: Data Preparation (20 mins)
*   **Templates**: Where to find them, how to use them.
*   **Rules**: Dropdowns, Required Fields, SKU logic.
*   **Exercise**: Create a valid Excel file with 10 items.

### Module 3: Handling Errors (15 mins)
*   **Simulation**: Import a file with known errors.
*   **Fixing**: Reading the error log, fixing Excel, re-importing.
*   **Q&A**: Common pitfalls.

## Hands-on Exercise
1.  Download Base Template.
2.  Add 3 items: "Cotton 60s", "Silk Satin", "Poly Drill".
3.  Import them.
4.  Verify in Fabric List.

## Certification
*   User must successfully import a batch of 20 items without errors to be "Certified".
