# Customer Support Guide

## Common User Issues

**Q: I can't see the "Export" button.**
*   **A**: Ensure you are logged in with a Manager or Admin role. Viewer roles cannot export data.

**Q: My uploaded design isn't showing.**
*   **A**: Check the `Pending Information` dashboard. The design might be uploaded but not linked to a fabric yet.

**Q: WhatsApp messages aren't going through.**
*   **A**: The 24-hour session window might have closed. You can only send "Template" messages to initiate conversation after 24 hours.

## Escalation
If a technical error persists (e.g., "500 Internal Server Error"), take a screenshot and forward it to the IT Admin with the time of occurrence.
