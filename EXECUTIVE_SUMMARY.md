## 4. Quick Start
1.  **Environment**: Copy `.env.example` to `.env` and fill in credentials.
2.  **Install**: Run `npm install` to get dependencies.
3.  **Run**: Start the server with `npm run dev`.
4.  **Test**: Navigate to `/admin/whatsapp-inbox` to verify connectivity.
