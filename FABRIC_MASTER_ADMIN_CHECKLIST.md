# Fabric Master Admin Checklist

## Daily Tasks
- [ ] **Review New Uploads**: Check the "Pending Info" dashboard for incomplete design uploads.
- [ ] **Sync Stock**: Verify stock levels for "Star" fabrics are accurate.

## Weekly Tasks
- [ ] **Archive Old Patterns**: Mark seasonal designs as "Inactive".
- [ ] **Review Suppliers**: Ensure active fabrics have valid supplier links.

## Monthly Tasks
- [ ] **Cleanup**: Delete temporary or test fabrics created during the month.
- [ ] **Performance**: Check if the dashboard loads quickly; archive old data if slow.

## Quarterly Tasks
- [ ] **Pricing Update**: Review Mill Costs and update Base Fabric rates.
- [ ] **Audit**: Cross-reference physical inventory with Fabric Master counts.
