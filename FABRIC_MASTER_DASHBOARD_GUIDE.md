# Fabric Master Dashboard Guide

## Overview
The **Fabric Master Dashboard** is your central command center for managing the entire textile catalog. It has been redesigned with a modern, professional interface to make managing thousands of fabrics efficient and intuitive.

## Key Features

### 1. Unified Dashboard Structure
The dashboard is divided into three main tabs:
- **Base Fabric**: The raw materials (e.g., Cotton, Rayon).
- **Finish Fabric**: Processed versions (e.g., Printed Rayon).
- **Fancy Finish**: Value-added versions (e.g., Embroidered Rayon).

### 2. Smart Grouping
Instead of a long list, fabrics are intelligently grouped:
- **Base Fabrics**: Grouped by material type (Cotton vs. Rayon).
- **Finish Fabrics**: Grouped by Process (Printed vs. Solid).
- **Fancy Fabrics**: Grouped by Value Addition (Embroidery vs. Handwork).

*Tip: Click the group header to expand/collapse sections.*

### 3. Bulk Actions Toolbar
Select multiple items using checkboxes to activate the **floating toolbar** at the bottom of the screen.
- **Star/Unstar**: Quickly mark favorites for easy access later.
- **Mark Active/Inactive**: Bulk update status without opening each item.
- **Delete**: Remove multiple items at once.

### 4. Advanced Filtering
The persistent **Filter Bar** gives you quick access to search results counts and active filters.
- Click **Advanced Filters** to open detailed options (Width, GSM, Process Type).
- Use **Clear All** to reset everything instantly.

### 5. Statistics at a Glance
Four statistics cards at the top of each dashboard show:
- Total Fabrics
- Starred Count (Favorites)
- Active Count (In-stock/Sellable)
- Inactive Count (Archived)

## Best Practices

### Organizing Your Catalog
1.  **Start with Base Fabrics**: Ensure your raw materials are defined correctly with proper HSN codes.
2.  **Create Finish Fabrics**: Use the "New from Base" button to create derived fabrics. This maintains the lineage so you know that "Printed Rayon 14kg" comes from "Rayon 14kg".
3.  **Use Stars**: Star your most popular or currently trending fabrics to keep them at the top of lists.

### Bulk Cleanup
1.  Use filters to find old/inactive stock (e.g., Filter Status = "Inactive").
2.  Select All items in the visible list.
3.  Use the **Delete** button in the bulk toolbar.
4.  **Safety Check**: The system will warn you if any selected fabrics are being used in Designs or Orders before deleting.

### Search Tips
- You can search by **Fabric Name** or **HSN Code**.
- Search is instant and filters the list in real-time.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| **"Delete Failed"** | The fabric likely has dependent items (e.g., Designs). Check the warning in the delete modal. You may need to delete the designs first or check "Delete Dependencies". |
| **"No Items Found"** | Check your active filters. Click "Clear All" in the filter bar. |
| **"Loading..." stuck** | Check your internet connection. Refresh the page. |

## Shortcuts
- **Click Row**: Expands details (for Finish/Fancy fabrics).
- **Click Star**: Toggles favorite status instantly.
- **Click Badge**: Clicking a "Designs" badge opens the design manager for that fabric.
