# ⭐ Fabric Master Features

## Core Features
*   **CRUD Operations**: Full Create, Read, Update, Delete capabilities.
*   **Auto-Generation**: Intelligent logic automatically generates standardized SKUs and Names based on specifications.
*   **Real-time Preview**: Immediate visual feedback during data entry.
*   **Validation**: Robust checks preventing invalid data entry (e.g., negative weights).

## List Features
*   **Grouped View**: Organized hierarchy by Base material.
*   **Advanced Filtering**: Multi-parameter filtering (Base + Finish + Width).
*   **Sortable Columns**: Click headers to sort by GSM, Name, etc.
*   **Sticky Interface**: Headers and key columns stay visible during scroll.

## Bulk Features
*   **Bulk Delete**: Remove multiple obsolete records safely.
*   **Excel Export**: One-click download of filtered or selected data.
*   **Batch Selection**: "Select All" capabilities for groups.

## Security Features
*   **Role-Based Access**: Only authorized staff can edit master data.
*   **Dependency Checks**: Prevents deletion of fabrics currently in use by orders.
*   **Audit Trail**: (Backend) Tracks creation and modification timestamps.
