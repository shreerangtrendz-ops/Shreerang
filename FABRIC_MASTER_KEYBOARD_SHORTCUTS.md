# ⌨️ FABRIC MASTER KEYBOARD SHORTCUTS

This guide outlines useful keyboard shortcuts to enhance productivity within the Fabric Master module.

---

## 1. Navigation Shortcuts

| Shortcut    | Action                      | Context                  |
| :---------- | :-------------------------- | :----------------------- |
| `Ctrl + N`  | Create New Fabric           | Any page                 |
| `Ctrl + F`  | Focus Search/Filter         | Fabric List Page         |
| `Esc`       | Clear Search/Close Dialog   | Fabric List Page         |
| `Alt + L`   | Go to Fabric List           | Any Fabric Form Page     |

---

## 2. Form Shortcuts (Create/Edit Fabric)

| Shortcut    | Action                              | Context                  |
| :---------- | :---------------------------------- | :----------------------- |
| `Ctrl + S`  | Save Fabric                         | Create/Edit Fabric Form  |
| `Alt + S`   | Save Fabric (Alternative)           | Create/Edit Fabric Form  |
| `Esc`       | Cancel / Go Back                    | Create/Edit Fabric Form  |
| `Tab`       | Next Field                          | Create/Edit Fabric Form  |
| `Shift + Tab`| Previous Field                     | Create/Edit Fabric Form  |

---

## 3. Table Shortcuts (Fabric List Page)

| Shortcut      | Action                          | Context                  |
| :------------ | :------------------------------ | :----------------------- |
| `Ctrl + A`    | Select All Visible Rows         | Fabric List Table        |
| `Delete`      | Delete Selected Fabrics         | Fabric List Table        |
| `Ctrl + E`    | Export Selected to Excel        | Fabric List Table        |
| `Arrow Up/Down`| Navigate Table Rows             | Fabric List Table        |
| `Enter`       | Open Edit Form for Selected Row | Fabric List Table (on row focus) |

---

## 4. Search Shortcuts (Fabric List Page)

| Shortcut    | Action                      | Context                  |
| :---------- | :-------------------------- | :----------------------- |
| `Ctrl + F`  | Focus Search Input          | Fabric List Page         |
| `Enter`     | Apply Search Filter         | Search Input Field       |
| `Esc`       | Clear Search Input          | Search Input Field       |

---

**Note**: Keyboard shortcuts are designed for efficiency. Browser-specific shortcuts (e.g., `Ctrl+F` for page search) might override some of these.
