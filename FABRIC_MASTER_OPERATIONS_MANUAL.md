# Fabric Master Operations Manual

## 1. Daily Operations
*   **Start of Day**: Check dashboard stats. Ensure "Active" counts match warehouse targets.
*   **During Day**: Production team adds new Finish Fabrics as they are processed.
*   **End of Day**: Manager reviews new entries for naming accuracy.

## 2. Weekly Maintenance
*   **Monday**: Review "Starred" items. Unstar fabrics that are no longer in high rotation.
*   **Friday**: Check for "Inactive" fabrics that can be permanently archived/deleted.

## 3. Emergency Procedures
*   **Accidental Delete**: Contact Admin immediately. Database Point-in-Time Recovery (PITR) is available for 7 days.
*   **System Down**: Use the offline "Emergency Fabric Spreadsheet" until system is restored.

## 4. Support
*   **Internal**: Contact IT Admin (admin@company.com).
*   **External**: Refer to Supabase status page for outage info.
