# Fabric Master Quick Start

## ⚡ 5-Minute Setup
1. **Navigate**: Go to **Fabric Master** > **New Fabric**.
2. **Select**: Click a **Fabric Image** (e.g., "60x60 Cotton").
3. **Width**: Select a **Width** (e.g., "58").
4. **Submit**: Click **"Add Fabric to Master"**.
5. **Done**: You have created a fabric with a generated SKU!

## 📋 Common Tasks

| Task | Steps |
| :--- | :--- |
| **Add Single Fabric** | Click Image -> Select Width -> Submit |
| **Add Custom Fabric** | Skip Image -> Type Name -> Select Width/Base -> Submit |
| **Modify Details** | Click Image -> Edit "GSM" or "Weight" fields -> Submit |

## ⌨️ Keyboard Shortcuts
*   **Tab**: Move to next field.
*   **Space**: Select the focused image card.
*   **Enter**: Submit form (when on submit button).
*   **Esc**: Cancel/Close.

## 🔍 Quick Reference Table
| Field | Status | Auto-Filled? | Format |
| :--- | :--- | :--- | :--- |
| **Fabric Name** | Required | Yes | Text |
| **Width** | Required | No | Dropdown |
| **Base** | Required | Yes | Dropdown |
| **Finish** | Required | Yes (Default Greige) | Dropdown |
| **SKU** | Auto | Yes | Width-Code-Finish |

## ⚠️ Common Errors
*   **Validation Error**: "Width is required" -> Select a width from the dropdown.
*   **Duplicate**: "SKU already exists" -> Change the Short Code or Fabric Name.
