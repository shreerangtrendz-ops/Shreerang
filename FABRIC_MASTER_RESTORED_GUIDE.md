# Fabric Master (Restored & Enhanced)

## Intelligent Organization
The Fabric Master is now split into three distinct sections for better clarity:

1.  **Base Fabrics**: Organized by Material (Cotton, Rayon) -> Specs.
2.  **Finish Fabrics**: Organized by Process (Print, Dye) -> Base.
3.  **Fancy Finish**: Organized by Value Addition -> Parent Finish.

## New Features
*   **Grouping:** Lists are now grouped automatically to reduce clutter.
*   **Favorites:** Star your most used fabrics to keep them at the top.
*   **Bulk Actions:** Select multiple items to Delete, Star, or Change Status.
*   **Dependency Tracking:** The system alerts you if you try to delete a fabric that is being used by another (e.g. deleting a Base Fabric used by a Finish Fabric).

## Workflow
1.  Create **Base Fabric** (Raw Material).
2.  Create **Finish Fabric** linked to Base (Processed).
3.  Create **Fancy Finish** linked to Finish (Value Added).
4.  Upload **Designs** to Finish/Fancy fabrics.
