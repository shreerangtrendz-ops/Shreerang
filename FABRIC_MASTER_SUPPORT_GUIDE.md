# Fabric Master Support Guide

## Common User Questions

**Q: I can't find a fabric I just added.**
A: Check if you have any filters active (top bar). Also check if you are in the correct tab (Base vs Finish).

**Q: Why can't I delete this fabric?**
A: It is likely being used in a Design or Order. You must delete or delink those items first.

**Q: How do I print a list of fabrics?**
A: Use the "Export to Excel" feature (coming soon) or print the browser page (Ctrl+P).
