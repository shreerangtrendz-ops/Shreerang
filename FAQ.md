# Frequently Asked Questions (FAQ)

### General
**Q: What browser should I use?**
A: Chrome, Firefox, Safari, or Edge (latest versions).

**Q: Can I use this on mobile?**
A: Yes, the admin panel is responsive, but desktop is recommended for complex data entry like Costing.

### Fabric Management
**Q: How do I edit a fabric?**
A: Go to the Fabric List, find the row, click the **Edit** (pencil) icon.

**Q: How do I delete a fabric?**
A: Click the **Delete** (trash) icon. *Note: You cannot delete a fabric if it has linked prices or designs.*

### Pricing
**Q: Can I see old prices?**
A: Yes, the "History" button in the Pricing table shows a log of all effective dates and prices.

**Q: How do I update a price?**
A: Add a *new* price entry with today's date. The system automatically treats the entry with the latest effective date as the current price.

### Data & Integration
**Q: How do I export data?**
A: Go to **Bulk Import/Export**, select the "Export" tab, choose your criteria, and download as CSV.

**Q: Where are images stored?**
A: Images are stored securely on Bunny.net CDN, not in the database, to keep the system fast.

**Q: Is my data backed up?**
A: Supabase handles daily database backups automatically.
