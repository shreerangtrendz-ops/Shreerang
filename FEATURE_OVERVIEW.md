# ✨ Feature Overview

## 1. Sales Order Form
**Overview**: A comprehensive form to create orders, linking customers, fabrics, and design assets in one flow.
**Key Features**:
*   Dynamic Item Rows (Add/Remove items).
*   **Integrated Design Upload**: Drag & drop designs directly into line items.
*   Auto-calculation of taxes and totals.
**Screenshot**: *Refer to Sales Order Image in documentation assets.*

## 2. WhatsApp Inbox
**Overview**: A CRM-style inbox embedded in the admin panel.
**Key Features**:
*   **Real-time**: Messages appear instantly.
*   **Context Aware**: See customer details and past orders while chatting.
*   **Rich Media**: Send images and documents.
**Screenshot**: *Refer to WhatsApp Inbox Image in documentation assets.*

## 3. Design Manager
**Overview**: Central repository for all fabric patterns and prints.
**Key Features**:
*   **Bunny.net CDN**: Fast global delivery.
*   **Tagging**: Categorize by Print Concept (e.g., Floral, Geometric).
*   **Search**: Find designs by auto-generated SKU.

## 4. Fabric Master
**Overview**: detailed specification management for fabrics.
**Key Features**:
*   **Grouping**: Base, Finish, and Fancy Finish hierarchies.
*   **Specs**: Detailed input for GSM, Width, Weave.
