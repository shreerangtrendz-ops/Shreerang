# GST Compliance Checklist

## System Configuration
- [ ] **Master Setup**: All process and value addition HSN codes are defined in `HSN Code Master`.
- [ ] **Base Fabrics**: Every Base Fabric record has a valid 4 or 6 digit HSN code and GST rate.
- [ ] **Garments**: Every Readymade Garment record has a valid HSN code and GST rate.

## Transaction Verification
- [ ] **Purchase Orders**: Verify that POs generated by the system display the correct Base Fabric HSN.
- [ ] **Job Work Orders**: Verify that Job Cards/Orders display the correct Process/Service HSN (e.g., 9988 or 5209).
- [ ] **Sales Invoices**: Verify that invoices generated for customers display the Fabric or Garment HSN and the breakdown of CGST/SGST/IGST.

## Data Integrity
- [ ] **Input Tax Credit**: Ensure that the GST claimed on purchases matches the HSN codes on the supplier's invoice.
- [ ] **Classification**: Ensure fabrics are not misclassified (e.g., mislabeling Synthetic as Cotton) which could lead to rate differences.
- [ ] **Rate Updates**: Have a process to update the HSN Master if the government changes GST notification rates.

## Record Keeping
- [ ] **Audit Trail**: Ensure all changes to HSN codes in the Master are logged (who changed it and when).
- [ ] **Documentation**: Maintain a digital or physical copy of the HSN Reference Table for staff.
- [ ] **Consistency**: Ensure the same product is not billed under two different HSN codes to different customers unless there is a valid reason (e.g., price-based rate change).
