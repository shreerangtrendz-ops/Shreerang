# Job Work Bill Module Guide

## Purpose
Tracks expenses related to value addition (Dyeing, Printing, Embroidery, etc.).

## Key Features
*   **Design Linking**: Costs are attributed to specific designs.
*   **Process Tracking**: Categorize by process type (e.g., "Embroidery").

## How to Add
1.  Click **"Add Job Work Bill"**.
2.  Select **Job Worker**.
3.  Enter **Design Number**.
4.  Enter **Process** (e.g., "Digital Print").
5.  Enter **Qty** and **Rate**.
6.  Save.

## Important
*   Unlike Purchase Bills, Job Work bills usually don't add stock quantity in the same way; they add *value* to existing stock.
*   Ensure the **Job Worker** selected matches the one assigned in the Fabric Master for accurate rate history.
