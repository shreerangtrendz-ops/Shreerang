# Purchase Bill Module Guide

## Purpose
Tracks all incoming invoices for raw materials (Base Fabric), semi-finished goods, and ready-made items.

## Key Features
*   **Brokerage Tracking**: Separate field for broker commission.
*   **Fabric Linking**: Links costs directly to inventory items.
*   **AI Import**: Upload image -> Extract Data -> Verify -> Save.

## How to Add
1.  Click **"Add Purchase Bill"**.
2.  Select **Supplier**.
3.  Select **Fabric Type** (Base/Finish/etc.).
4.  Enter **Item Name** (e.g., "60x60 Cotton").
5.  Enter **Qty** and **Rate**.
6.  (Optional) Enter **Brokerage %**.
7.  Save.

## Best Practices
*   Always enter the **HSN Code** for GST compliance.
*   Use the **Notes** field to record lot numbers or batch details.
