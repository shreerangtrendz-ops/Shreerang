# Quick Reference Guide

| Action | Navigation Path | Key Steps |
|--------|-----------------|-----------|
| **Add Base Fabric** | Fabric Master > Base > Add New | Enter Name, Width, GSM. **No Finish field**. |
| **Add Finish Fabric** | Fabric Master > Finish > Add New | Select Base. Select Process. **Define Finish here**. |
| **Import Excel** | Fabric Master > Import | Download Template -> Upload -> Map -> Import. |
| **Sync Drive** | Settings > Google Drive | Connect Account -> Select Folder -> Sync Now. |
| **Send WhatsApp** | Marketing > WhatsApp | Select Customer/Group -> Choose Template -> Send. |
| **Add Supplier** | People > Suppliers > Add | Enter Name, Type, GST. Link to fabrics if needed. |
| **Add Job Worker** | People > Job Workers > Add | Enter Name, Process Type (e.g. Dyeing), Rate. |
| **Check Pending Designs** | Design Management > Pending Info | Review list. Click "Edit" to add missing details. |
| **Bulk Delete** | Any Master List | Select Checkboxes -> Click Trash Icon in Toolbar. |
