# Fabric Management System

A comprehensive ERP solution for textile manufacturing, inventory, and sales management.

## Features
*   **Fabric Master**: Manage Base, Finish, and Fancy fabrics hierarchy.
*   **Design Center**: Visual gallery, combo pack builder, and stock management.
*   **Smart Orders**: Auto-fetching order entry system.
*   **Analytics**: Dashboards for sales and inventory tracking.

## Quick Start
1.  **Install**: `npm install`
2.  **Configure**: Copy `.env.example` to `.env` and add Supabase keys.
3.  **Run**: `npm run dev`
