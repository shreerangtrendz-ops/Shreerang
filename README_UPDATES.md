## 🌟 New Features (v1.0.0)
- **WhatsApp Integration**: Chat with customers directly from the admin panel.
- **Design Cloud**: Store unlimited fabric designs via Bunny.net CDN.
- **Smart Orders**: Attach designs to sales orders via drag-and-drop.

## 🚀 Quick Start
1.  `npm install`
2.  `cp .env.example .env` (Fill in credentials)
3.  `npm run dev`
