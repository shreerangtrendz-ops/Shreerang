# User Guide

## Getting Started
1.  **Login**: Access the admin panel using your credentials.
2.  **Dashboard**: The main dashboard gives you an overview of recent orders, WhatsApp messages, and stock alerts.
3.  **Sidebar**: Use the left sidebar to navigate between modules.

## WhatsApp Inbox Guide
Navigate to **WhatsApp Inbox** in the sidebar.

### Interface Overview
*   **Left Panel**: List of active conversations. Search by name or phone number.
*   **Middle Panel**: Chat thread. Shows message history.
*   **Right Panel**: Customer details and quick actions.

### Sending Messages
1.  Select a conversation.
2.  Type your message in the input box at the bottom.
3.  Click **Send** or press Enter.
4.  **Attachments**: Click the paperclip icon to send images or documents.
5.  **Quick Replies**: Type `/` to select a template.

## Design Upload Guide
Navigate to **Design Manager**.

### Uploading Designs
1.  Click **Upload Design** or drag files onto the drop zone.
2.  **Design Number**: The system auto-generates a number based on the filename. You can edit this.
3.  **Descriptions**: Select tags for:
    *   **Concept**: Print, Embroidery, etc.
    *   **Pattern**: Floral, Geometric, etc.
    *   **Value Addition**: Foil, Glitter, etc.
4.  Click **Save** to upload to the secure cloud.

## Sales Order Guide
Navigate to **Sales Orders**.

### Creating an Order
1.  Click **New Order**.
2.  **Customer**: Search and select an existing customer or create a new one.
3.  **Fabric Selection**: Choose the Base Fabric.
4.  **Design**: Select a design from the library (previously uploaded).
5.  **Quantity**: Enter meters/units.
6.  **Instructions**: Add specific notes (e.g., "Urgent delivery").
7.  Click **Submit Order**.

## Fabric Master Guide
Navigate to **Fabric Master**.

### Managing Fabrics
*   **Add Fabric**: Click the "+" button. Fill in GSM, Width, and Construction details.
*   **Edit**: Click the pencil icon on any fabric row.
*   **Star**: Click the star icon to mark a fabric as a favorite for quick access.

## Common Tasks

### Link Order to WhatsApp
1.  In the WhatsApp Inbox, open a conversation.
2.  In the right panel, click **Link Order**.
3.  Select the active Sales Order.
4.  Now, order updates can be sent directly to this chat.

### Bulk Import Fabrics
1.  Go to Fabric Master.
2.  Click **Import**.
3.  Download the **Excel Template**.
4.  Fill in your data and upload the file.
