# User Training Guide

## 1. Getting Started
- **Login**: Use your assigned email and password.
- **Dashboard**: Your home base showing recent activity and quick links.

## 2. Managing Fabrics
- **Adding Fabric**: Always start with **Base Fabric**. If it exists, proceed to **Finish Fabric**.
- **Naming**: The system auto-generates names to ensure consistency. Do not manually override unless necessary.

## 3. Uploading Designs
- **Method A**: If you have one photo for a specific fabric, go to that Fabric's page.
- **Method B**: If you have 100 photos from a photoshoot, use **Bulk Upload**. The system reads filenames like "D-101.jpg" to auto-fill the Design Number.

## 4. Best Practices
- **Images**: Always use JPG/PNG. The system will auto-compress them.
- **Search**: Use the global search or filters before creating a new item to avoid duplicates.
- **Data Entry**: Fill as many optional fields as possible (GSM, Width) to help with filtering later.
