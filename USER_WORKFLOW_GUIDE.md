# User Workflow Guide: Fabric Master

## 1. Quick Start (5 Minutes)
1.  Go to **Fabric Master > New Fabric**.
2.  Click a **Fabric Image** (e.g., "60x60 Cotton").
3.  Select a **Width** (e.g., "58").
4.  Click **Submit**.
**Done!** You've added a new fabric.

## 2. Detailed Workflow

### Step 1: Navigate
Go to the Admin Dashboard, select **Fabric & Design**, then **Fabric Master**, and click **+ New Fabric**.

### Step 2: View Images
You will see a grid of 7 common fabric types at the top. These are shortcuts to help you fill the form faster.

### Step 3: Select Fabric
Click on a card (e.g., **"Poly Crepe"**). It will highlight with a blue border and checkmark.
*   *Result*: The "Fabric Name" field automatically fills with "Poly Crepe", and "Base" fills with "Polyester".

### Step 4: Verify Fields
Scroll down to the **Basic Information** section. Notice that fields are already filled.

### Step 5: Select Width
The **Width** field is required but cannot be auto-guessed. Click the dropdown and select a width (e.g., **44"**).

### Step 6: Verify SKU
Look at the **Live Preview** box on the right. The SKU will change to something like `44-POLY-CRP-GRG`.

### Step 7: Change Finish (Optional)
If this fabric is not "Greige" (Raw), change the **Finish** dropdown to **"RFD"** or **"Dyed"**. Watch the SKU update.

### Step 8: Technical Details (Optional)
Fill in **GSM**, **Weight**, or **Yarn Count** if you have that information. These are optional but good for records.

### Step 9: Review
Check the **Base Fabric Name** and **SKU** in the preview box. Does it look correct?

### Step 10: Submit
Click the **"Add Fabric to Master"** button at the bottom right.

### Step 11: Success
A green message "Fabric added successfully" will appear at the top right.

### Step 12: Repeat
The form clears itself. You can now click another image to add the next fabric immediately.

## 3. Field Reference
*   **Fabric Name**: Commercial name.
*   **Width**: Usable width of the fabric roll.
*   **Base**: Main material (Cotton, Poly, etc.).
*   **Finish**: Processing state (Greige, RFD, etc.).
*   **GSM**: Grams per Square Meter (Density).
*   **Weight**: Kg per meter/yard.

## 4. SKU Format Explanation
Format: `[Width]-[ShortCode]-[Finish]`
*   **Width**: Number (e.g., 58).
*   **ShortCode**: 3-4 letters representing the fabric (e.g., COT for Cotton).
*   **Finish**: 3 letters (GRG=Greige, RFD=Ready for Dye, DYD=Dyed).

## 5. Tips & Best Practices
*   **Use Images**: Always start by clicking an image if possible. It saves typing.
*   **Check Widths**: Always double-check the width. It's the most common mistake.
*   **Leave Short Code**: Let the AI generate the short code unless you have a specific internal code you must use.

## 6. Common Scenarios
*   **Custom Fabric**: If your fabric isn't in the images, just skip the images and type the name manually.
*   **Duplicate**: If you try to add the exact same fabric twice, the system might warn you about a duplicate SKU. Change the name slightly or check if it already exists.

## 7. Troubleshooting
*   **SKU is "XX-XXX-XXX"**: This means you haven't filled the required fields (Name, Width, Finish) yet.
*   **Submit Button Doesn't Work**: Look for red error text under a field (usually Width).
