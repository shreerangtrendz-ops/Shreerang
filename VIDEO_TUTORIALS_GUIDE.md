# Video Tutorials Guide

This guide outlines the scripts and flow for the training video series.

## Tutorial 1: System Overview (5 min)
-   **Intro**: Welcome to Schiffli Fabric Master.
-   **Dashboard**: Walkthrough of the 4 main KPI cards.
-   **Sidebar**: Navigation overview.

## Tutorial 2: Creating Your First Base Fabric (8 min)
-   **Step 1**: Go to Fabric Master.
-   **Step 2**: Click Add Fabric.
-   **Step 3**: Explain fields (SKU auto-generation, Width, GSM).
-   **Step 4**: Save and Verify in List.

## Tutorial 3: Managing Fabric Prices (7 min)
-   **Concept**: Explain "Effective Date".
-   **Action**: Add a price change for a fabric.
-   **View**: Show how the History table updates.

## Tutorial 4: Uploading Designs (6 min)
-   **Action**: Drag and drop an image.
-   **Integration**: Explain how Bunny.net URL is generated.
-   **Link**: Associate design with a Fabric ID.

## Tutorial 5: Schiffli Costing (9 min)
-   **Scenario**: Calculating cost for a 500m order.
-   **Inputs**: Enter Grey Rate, Job Rate, Shortage.
-   **Result**: Interpret the "Final Cost per Meter".
